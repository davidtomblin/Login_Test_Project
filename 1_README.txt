# Swag Labs Login Test Project

## Project Overview
This project demonstrates manual QA testing of the login functionality on the Swag Labs demo site (https://www.saucedemo.com). It includes 10 test cases covering positive and negative login scenarios, as well as a documented bug report.

## Contents
- **2_TestPlan.md** – Contains the objective, scope, in-scope/out-of-scope features, and testing approach.  

- **3_Test_Cases.xlsx** – 10 manual test cases with Expected Results, Actual Results, and Pass/Fail status.  

- **4_bug_report.md** – Contains Markdown files for identified bugs. Currently includes:
  - `TC007_LoginButtonEnabled.md` – Login button enabled when only one field is filled.  
- **screenshots/** 5_Bug_Report_TC007_Screenshot – Screenshot of bug reproduction.

## Objectives
- Validate that login functionality works as expected.
- Test input validation, error messages, and edge cases.
- Document findings in a professional QA format.

## Instructions
1. Open `3_TestCases.xlsx` to view and review all test cases and results.  
2. Review `bug_report/` folder for detailed bug documentation.  
3. Refer to screenshots for visual confirmation of test execution or bug reproduction.
